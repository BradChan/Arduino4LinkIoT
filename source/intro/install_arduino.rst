==================================
安装Arduino IDE
==================================

下载Arduino IDE
================

首先需要打开Arduino官网，并根据自己所用电脑的系统，`下载最新版Arduino IDE`_ 。

https://www.arduino.cc/en/Main/Software

Adruino IDE支持Windows XP及以后的系统、macOS 10.8及以后的系统、Linux等系统。

.. _下载最新版Arduino IDE: https://www.arduino.cc/en/Main/Software
